# Central Bank of Kenya - Revenue & Expenditure Analysis

## 📊 Project Overview
This project analyzes the Central Bank of Kenya's Revenue and Expenditure data from 2020-2024, extracted from the CBK Statistical Bulletin (June 2024). The analysis includes exploratory data analysis, trend identification, and predictive modeling.

**Author:** Philip Tait  

---

## 🎯 Objectives
1. Extract and clean government finance data from CBK Statistical Bulletin
2. Analyze revenue and expenditure trends (2020-2024)
3. Identify key patterns and insights in Kenya's fiscal position
4. Develop predictive models for revenue/expenditure forecasting
5. Present findings in a professional PowerPoint presentation

## 🚀 How to Run This Project

### Prerequisites
- Python 3.10 or higher
- Java (required for tabula-py)
- Jupyter Notebook

### Installation
```bash
# Clone or download the project
cd CBK_Revenue_Expenditure_Analysis

# Install dependencies
pip install -r requirements.txt

# Launch Jupyter Notebook
jupyter notebook
```

### Run Analysis
1. Start with `notebooks/01_data_extraction.ipynb`
2. Proceed sequentially through notebooks
3. Review outputs in `reports/` folder
4. View presentation in `presentation/` folder
